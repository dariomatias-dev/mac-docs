## Símbolos

| Símbolo           | Significado                     |
| ----------------- | ------------------------------- |
| $\forall$         | para todo                       |
| $\exists$         | existe pelo menos um            |
| $\in$             | pertence                        |
| $\Rightarrow$     | se... então (implica)           |
| $\Leftrightarrow$ | se e somente se (bicondicional) |

Por exemplo, $A \subseteq B$ e $B \subseteq A \Leftrightarrow A = B$.

## Conjunto vazio

É o conjunto que não possui nenhum elemento, denotado por $\varnothing$ ou
$\{\,\}$.

$$
B = \{x \mid x \in \mathbb{N} \text{ e } x < 0\} = \varnothing
$$

## Linguagem dos conjuntos

Sejam $A$ e $B$ conjuntos. Dizemos que $A$ está contido em $B$ quando todo
elemento de $A$ também é elemento de $B$:

$$
(\forall x)(x \in A \Rightarrow x \in B)
$$

### Encontrando a fórmula do termo geral

Para descrever uma sequência por uma fórmula, usamos funções polinomiais e
determinamos os coeficientes:

- **1.º grau:** $f(x) = ax + b$
- **2.º grau:** $f(x) = ax^{2} + bx + c$
- **3.º grau:** $f(x) = ax^{3} + bx^{2} + cx + d$

<Exercise>
Escreva $B = \{-1, 0, 3, 8, 15, \ldots\}$ na forma de conjunto.

<Answer>
Supondo grau $2$, $f(x) = ax^{2} + bx + c$ com $x = 0, 1, 2, \ldots$:

$$
\begin{aligned}
f(0) &= c = -1 \\
f(1) &= a + b + c = 0 \;\Rightarrow\; a + b = 1 \\
f(2) &= 4a + 2b + c = 3 \;\Rightarrow\; 4a + 2b = 4
\end{aligned}
$$

Resolvendo: $a = 1$, $b = 0$, $c = -1$, logo $f(x) = x^{2} - 1$.

$$
B = \{x \mid x = y^{2} - 1,\ y \in \mathbb{Z}_{+}\}
$$

</Answer>
</Exercise>

<Exercise>
Escreva $C = \{2, 3, 10, 29, \ldots\}$ na forma de conjunto.

<Answer>
Supondo grau $3$, $f(x) = ax^{3} + bx^{2} + cx + d$:

$$
\begin{aligned}
f(0) &= d = 2 \\
f(1) &= a + b + c + d = 3 \\
f(2) &= 8a + 4b + 2c + d = 10 \\
f(3) &= 27a + 9b + 3c + d = 29
\end{aligned}
$$

Resolvendo: $a = 1$, $b = 0$, $c = 0$, $d = 2$, logo $f(x) = x^{3} + 2$.

$$
C = \{x \mid x = y^{3} + 2,\ y \in \mathbb{Z}_{+}\}
$$

</Answer>
</Exercise>
