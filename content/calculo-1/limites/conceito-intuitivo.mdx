Intuitivamente, dizer que $\lim_{x \to a} f(x) = L$ significa que podemos deixar
$f(x)$ tão próximo de $L$ quanto quisermos, bastando tomar $x$ suficientemente
próximo de $a$ — sem precisar que $f$ esteja definida em $a$.

## Aproximação numérica

Observe $f(x) = \dfrac{x^2 - 1}{x - 1}$ perto de $x = 1$:

| $x$  | $f(x)$ |
| ---- | ------ |
| 0,9  | 1,9    |
| 0,99 | 1,99   |
| 1,01 | 2,01   |
| 1,1  | 2,1    |

Os valores se acumulam em torno de $2$, logo $\lim_{x \to 1} f(x) = 2$.

## Limites laterais

O limite existe apenas quando os dois lados coincidem:

$$
\lim_{x \to a} f(x) = L \iff \lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L
$$

### Quando o limite não existe

Se os limites laterais diferem, o limite não existe. É o caso de $\lfloor x
\rfloor$ (piso) nos inteiros.

## Verificação numérica

Uma estimativa simples avalia $f$ dos dois lados de $a$ e tira a média:

```ts
function limiteAprox(f: (x: number) => number, a: number): number {
  const h = 1e-6;
  return (f(a - h) + f(a + h)) / 2;
}

limiteAprox((x) => (x * x - 1) / (x - 1), 1); // ~2
```
