---
title: Tabela de derivadas
description: Derivadas das funções mais comuns.
order: 1
---

| $f(x)$      | $f'(x)$                     |
| ----------- | --------------------------- |
| $c$         | $0$                         |
| $x^n$       | $n\,x^{n-1}$                |
| $e^x$       | $e^x$                       |
| $a^x$       | $a^x \ln a$                 |
| $\ln x$     | $\dfrac{1}{x}$              |
| $\sin x$    | $\cos x$                    |
| $\cos x$    | $-\sin x$                   |
| $\tan x$    | $\sec^2 x$                  |
| $\arcsin x$ | $\dfrac{1}{\sqrt{1 - x^2}}$ |
| $\arctan x$ | $\dfrac{1}{1 + x^2}$        |

<Callout type="connection" title="Conexão">
  Cada linha é a inversa de uma entrada na tabela de integrais imediatas.
</Callout>
